Vagrant配置参考

首先需要准备好Vbox管理器、Vagrant、Box

vagrant安装和启动

编辑Vagrantfile文件(后续再设置共享文件夹)

保存退出以后,在nginx 文件夹 使用

vagrant up
vagrant ssh

sudo su 可以提升权限为root

vagrant常用指令

vagrant box add box名 box路径（url或者本地路径）
vagrant box remove box名 销毁box
vagrant up 启动
vagrant halt 关闭
vagrant reload 重启
vagrant destroy 删除box
vagrant box list 查看box列表
